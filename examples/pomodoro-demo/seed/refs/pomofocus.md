# Pomofocus — 极简网页番茄钟

Pomofocus 是一个网页应用，界面干净到几乎没有东西：一个大数字倒计时、几个圆点标记第几个番茄、下面列一个今天的任务列表。不需要安装，打开浏览器就能用。免费版已经够用，付费版多一些主题和报告。

逻辑完全标准的 Pomodoro Technique：25/5/15，工作、短休、长休循环。任务可以估几个番茄、打勾完成。每天/每周有简单统计。

**我喜欢的**：轻量是最大优点，没有账号也能用，没有 push 通知骚扰你。颜色随工作/休息状态切换（红/绿/蓝），视觉状态一目了然。可以在旁边开着不占注意力。

**我不喜欢的**：太素了——我知道这是优点，但素到没有一点自己的性格，用起来像在用一个表单。任务列表功能很弱，没有层级、没有标签。休息界面什么都没有，让人不知道该干什么。本质上是一个静态页面，无法扩展或定制，想改什么只能 fork 自己搭。
