# Flow — 设计精致的 Mac 番茄钟

Flow 是我见过设计最用心的 macOS 番茄钟，没有之一。界面简洁但不寒酸，有种北欧设计感——大留白、圆形进度条、颜色柔和。App Store 评分一直很高，更新也勤快。菜单栏倒计时、专注模式（隐藏 dock 和通知）、深色/浅色模式，细节打磨得很舒服。

功能上中规中矩：自定义时长、任务列表、每日报告、HealthKit 集成（专注时长计入健康数据）。付费买断制，价格合理。

**我喜欢的**：那个圆形进度条动画丝滑，看着时间流逝不焦虑反而平静。配色方案挑得很准，说不出哪里好但就是耐看。进入专注模式时的过渡动画很小心，不突兀。

**我不喜欢的**：它的"精致"是一种很克制、很统一的精致，没有给用户留下任何"自己的"空间——我没法放一个小吉祥物在上面，没法让它变成我喜欢的那种像素感。统计数据还不够深，只看完成了多少个番茄，看不出专注质量。macOS 独占，没有网页版。
