番茄喵工程师
Personal Pomodoro Timer

Vision: A deeply personal pomodoro timer that captures the user's specific aesthetic (pixel-art 番茄喵 + lo-fi cozy vibes) and idiosyncratic rules (forced-walk 4th pomodoro, fattening cat progression, contemplative break messages) while maintaining all standard pomodoro functionality.

🍅 Core Pomodoro Features (Standard)

Timer Mechanics

  • 25-minute work cycles
  • 5-minute short breaks
  • 15-minute long break after 4 pomodoros
  • Visual countdown display
  • Clear work/break state indication

Today-Only Task Input

  • Single text input box
  • "Today's three things" format
  • Line breaks for multiple tasks
  • Collapsible/hideable task panel
  • No project management features

Sound System

  • True mute: When disabled, completely silent (no accidental sounds)
  • Optional timer completion chime
  • User-selectable sound from curated options
  • Binary state: ON or OFF (no volume adjustment)

🐱 Non-Standard Features (User's Personal Rules)

Pixel-Art 番茄喵 Mascot & Lo-Fi Aesthetic

Visual Design:

  • Central 16×16 or 32×32 pixel-art cat mascot
  • Orange cat head + tomato-red body
  • Squinting/content eyes, sitting pose
  • Lo-fi cozy color palette: warm cream background, brick red accents, amber countdown digits
  • Hand-crafted typography feel (avoid SF Pro sterility)
  • NO bright white backgrounds, NO SaaS blue colors

4th Pomodoro: Forced Walk Mode

Behavior: After completing the 4th pomodoro in a cycle, instead of normal break:

  • Entire UI switches to grayscale using filter: grayscale(1)
  • Display large text: "去走走，别带手机"
  • 番茄喵 becomes black & white, curled up sleeping
  • 10-minute countdown runs (still visible but desaturated)
  • "我回来了" button appears but rejects early clicks
  • Cat shakes head (small animation) if clicked too soon
  • Only after timer completes: button works, color returns, cat stretches awake

Progressive Cat Fattening System

Mechanics:

  • Cat has 5 sprite states: Normal → Slightly Chubby → Fat → Very Fat → Round Ball
  • Advances one level per completed 4-pomodoro cycle
  • Maxes out at "Round Ball" and stays there (achievement badge)
  • Resets to Normal each day (fresh motivation)
  • Visual progression serves as daily completion tracking

Custom Break Messages

Content: During 5-minute breaks, display ONE user-written contemplative message:

"窗户外面一直有鸟叫，我一直没认真听过。"
"手不知道什么时候开始握着的，松开。"
"喝点水，不是因为该喝，是因为想喝。"
  • NO motivational quotes from external libraries
  • NO "Stay focused!" type messaging
  • User pre-writes personal, introspective one-liners
  • Random selection from user's custom list
  • Large, centered text display
  • Cat shows in resting pose beside message

🛠 Technical Requirements

Architecture

  • No accounts, no cloud sync
  • Pure localStorage for all data persistence
  • Single-page web application
  • Works offline after initial load
  • Cross-browser compatible (modern browsers)

Data Storage

  • Timer states and current progress
  • Cat fatness level for current day
  • User's custom break messages array
  • Today's task input
  • Sound preference (on/off)
  • Auto-cleanup: reset daily progress at midnight

Performance

  • Lightweight pixel art assets (< 5KB total)
  • Smooth CSS animations for state transitions
  • Responsive design for desktop and tablet
  • 60fps timer updates without performance degradation

🎨 Asset Requirements

番茄喵 Sprite Set

  • 5 fatness levels (color versions)
  • 5 corresponding grayscale versions
  • Sleeping pose (for forced walk)
  • Stretching/waking animation frames
  • Head-shake rejection animation

Sound Assets

  • 2-3 gentle completion chimes
  • Lo-fi aesthetic (warm, non-jarring)
  • Short duration (< 2 seconds)
  • Small file sizes for web delivery

📱 User Experience Flow

Daily Session Start

  • User opens app, sees Normal-weight 番茄喵
  • Optional: enters today's tasks in collapsible panel
  • Clicks start, 25-minute timer begins
  • Cat sits contentedly, timer counts down in amber digits

Standard Cycle (Pomodoros 1-3)

  • Work session: 25 minutes, cat alert/focused
  • Break: 5 minutes, shows user's custom message + resting cat
  • Repeat until 4th pomodoro

4th Pomodoro Completion

  • Work completes → UI immediately goes grayscale
  • Cat curls up, goes black & white
  • 10-minute forced walk timer starts
  • After 10 minutes: "我回来了" button becomes active
  • Click → color returns, cat stretches, cycle completes
  • Cat advances to next fatness level

🚀 Success Metrics

  • Personal Fit: User continues using beyond 2-week mark (where other apps failed)
  • Rule Adherence: Forced walk feature actually encourages movement
  • Aesthetic Satisfaction: Lo-fi pixel art creates desired cozy atmosphere
  • Simplicity: No feature creep beyond core personal requirements
  • Reliability: Works consistently without bugs in timer logic
